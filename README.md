# Proyecto Expo-Fray

Proyecto realizado por 6to A - Informática en 2025 para la Expo-Fray.

---
## Tema: El uso del celular

Los alumnos debieron realizar una investigación sobre las *consecuencias* del **uso excesivo del celular** y que *buenos hábitos* desarrollar para mejorar nuestra salud y poder aprovechar bien el tiempo para otras cosas que también nos hacen bien.
Usando HTML y CSS crearon las distintas páginas pertinentes y luego fueron unidas teniendo una misma convención sobre los nombres de los archivos a la hora de enlazarlos.

Mediante el uso de JS, también se realizó un **test** para mostrar cuanto tiempo pasamos en un dia, semana, mes y año con el celular a base del ingresó de cuanto tiempo le dedicamos a cada app o red social.
